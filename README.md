## Getting Started

1. **Install dependencies:**

```bash
npm install
```

2. **Start the development server:**

```bash
npm start
```

3. **Build for production:**

```bash
npm run build
```

4. **Run linting**

```bash
npm run lint
```

5. **Run tests**

```bash
npm run test
```
